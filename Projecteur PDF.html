Projecteur PDF v1.6

Concepteur : Nguyen PHAM

Double-clic : Créer une note ($...$ pour LaTeX).

Alt + Clic sur note : Supprimer la note.

NAVIGATION & SLOTS :
• 1 à 9 : Charger un slot | Alt + Chiffre : Sauvegarder
• Tab : Changer de groupe (Classe)
• / : Sauvegarde rapide sur le slot actif
• + / - : Zoomer / Dézoomer

GESTION DU MASQUE :
• 0 : Alterner Plein écran / Masque personnalisé
• Maj + Flèches : Déplacer le coin Haut-Gauche
• Alt + Flèches : Déplacer le coin Bas-Droit

AFFICHAGE :
• * : Inverser les couleurs (Mode sombre)
